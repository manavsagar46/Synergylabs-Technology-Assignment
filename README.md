# User Management App

A simple CRUD application built with **React + Vite + Tailwind CSS** using **Context API** for global state management.  
Fetches users from **JSONPlaceholder**, with simulated Create, Edit, and Delete operations.  
Includes a fully responsive UI, smooth navigation via **React Router**, and global loading/error handling.  
Clean and modular code structure with reusable components for easy maintenance.  
Run locally using: `npm install` followed by `npm run dev`.
